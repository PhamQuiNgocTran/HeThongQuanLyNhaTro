# Hệ Thống Quản Lý Nhà Trọ

Dự án thực hành quản lý dự án bằng Jira, Figma và GitHub.

## Cấu trúc thư mục

- `api`: Chứa mã nguồn backend Node.js
- `database`: Chứa file SQL khởi tạo cơ sở dữ liệu MySQL
- `figma`: Chứa link thiết kế giao diện Figma
- `solution`: Chứa giao diện HTML, CSS, JavaScript
